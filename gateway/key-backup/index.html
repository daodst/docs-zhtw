    密钥备份和恢复

    用戶可以選擇將閘道的金鑰，驗證器金鑰和app加密檔備份，以供需要時恢復

    備份分為手動備份與上鏈備份，上鏈備份只需花費少量的gas，即可備份到鏈上，隨時可以下載

    手动备份

    點擊左側閘道選項，點擊[下載密鑰檔案]按鈕，將其保存到指定位置即可

    上链备份

    點擊左側閘道選項，點擊[閘道密鑰鏈上存儲]按鈕，輸入密碼，待上鏈成功即存儲完畢

    还原

    如果是鏈上存儲的，那麼[閘道密鑰鏈上存儲按鈕]按鈕會變更為[下載鏈上密鑰]，點擊後，會將三個檔案保存在閘道目錄下的GatewayKey目錄

    GatewayKey/
├── fmc.jks                 # app的簽名檔案
│── node_key.json           # 閘道密鑰檔案
└── priv_validator_key.json # 驗證器私密密鑰檔案

    在鏈停止狀態下 將fmc.jks移動到家目錄下

    linux系統中 通過

    echo $HOME

    命令查看當前用戶家目錄位置

    windows系統中，打開資源管理器，在地址欄中輸入

    %homepath%

    即可進入當前用戶家目錄下

    將node_ key.json和priv_ validator_ key.json移動到gateway目錄下的 .stcd/config/ 下

    如果是手動存儲，則同樣的，將對應檔案還原到對應位置即可
